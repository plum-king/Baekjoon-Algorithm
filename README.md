- deque 라이브러리 사용
    - import 방법: from collections import deque
    - deque 생성: `dec = deque()`
    - 사용 가능한 method: deque.append(), deque.pop(), deque.popleft() 등
        - deque.pop(): 오른쪽 인자 가져와서 삭제
        - deque.popleft(): 왼쪽 인자 가져와서 삭제
        - deque.appendleft(item): 왼쪽 끝에 item 추가
